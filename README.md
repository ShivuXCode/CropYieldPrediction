# Parkinson Disease Detection
[![forthebadge made-with-python](http://ForTheBadge.com/images/badges/made-with-python.svg)](https://www.python.org/)<br>

This project applies machine learning to identify Parkinson's Disease from the provided dataset. It covers the full workflow from data preparation to model comparison, including:

1. Data collection
2. Data preprocessing
3. Exploratory data analysis
4. Dataset balancing and feature scaling
5. Training and evaluating multiple machine learning models

## Models Trained and Evaluated

The following classifiers were tested in this project:

1. Decision Tree Classifier
2. Random Forest Classifier
3. Logistic Regression
4. Support Vector Machine Classifier
5. Naive Bayes Classifier
6. K Nearest Neighbor Classifier
7. XGBoost Classifier

## Best Model

The strongest overall result came from the Random Forest Classifier, which achieved:

- Accuracy: 0.996102
- F1 Score: 0.961538
- R2 Score: 0.862471
